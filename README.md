# Taj-Mehel-Afron-Market

Afro-Asian grocery store website for **Taj Mehel Afro Asian Market** — Tikkuraitti 13, 01300 Vantaa, Finland.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- [Next.js](https://nextjs.org/)
- [Mantine](https://mantine.dev/)
- TypeScript
